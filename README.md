# REAL-TIME-BUS-TRACKING
A full-stack real-time bus tracking system that allows users to monitor live bus locations, routes, and estimated arrival updates through an interactive frontend interface.

Features:
->Real-time bus location tracking
->Live location updates
->Interactive frontend dashboard
->Route and bus monitoring
->Backend API integration
->Scalable project structure
->Responsive UI


Tech Stack:
Frontend:
->React.js
->JavaScript
->CSS


Backend:
->Node.js 
->REST APIs
->WebSocket / Socket.IO


Database:
->MongoDB 


Project Structure:

REAL-TIME-BUS-TRACKING:
│
├── backend
│
├── frontend
│
└── README.md


Installation:
->Clone Repository:
git clone https://github.com/ARI-004/REAL-TIME-BUS-TRACKING.git
->cd REAL-TIME-BUS-TRACKING


Frontend Setup
->cd frontend
->npm install
->npm run dev



Frontend runs on:
->http://localhost:5173


Backend Setup:
->cd backend
->npm install
->npm start



Backend runs on:
->http://localhost:5000


Use Cases:
->Public transport monitoring
->Smart city transportation systems
